# KasKelas 📊

![KasKelas Banner](kaskelas_banner_1777860129745.png)

**KasKelas** adalah aplikasi manajemen keuangan kas kelas yang modern, ringan, dan mudah digunakan. Dirancang khusus untuk membantu bendahara kelas mengelola iuran, pemasukan, dan pengeluaran secara transparan dan terorganisir.

---

## ✨ Fitur Unggulan

- 💎 **Modern UI/UX**: Desain premium dengan estetika *Glassmorphism* dan tema gelap yang nyaman di mata.
- 📈 **Visualisasi Data**: Pantau grafik pemasukan dan pengeluaran secara real-time menggunakan integrasi *MPAndroidChart*.
- 💾 **Penyimpanan Lokal**: Menggunakan *Room Database* untuk memastikan data aman dan dapat diakses tanpa koneksi internet.
- 👥 **Manajemen Anggota**: Kelola daftar anggota dan status pembayaran iuran dengan mudah.
- 📑 **Laporan Otomatis**: Ringkasan transaksi harian dan bulanan yang rapi.
- 📖 **Buku Panduan**: Tersedia [Manual Book](manual_book.html) interaktif di dalam aplikasi.

---

## 🚀 Cara Instalasi

1. **Download APK**: Pergi ke tab [Releases](https://github.com/ZeroMysty/kaskelas/releases) dan unduh versi terbaru.
2. **Izinkan Instalasi**: Aktifkan izin "Install from Unknown Sources" di pengaturan Android Anda.
3. **Instal & Jalankan**: Buka file APK dan ikuti instruksi instalasi.

---

## 🛠️ Tech Stack

- **Language**: Kotlin
- **Persistence**: Room Database
- **UI Components**: Material Design 3, Constraint Layout
- **Graphics**: MPAndroidChart
- **Architecture**: MVVM (Model-View-ViewModel)
